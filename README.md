# Personal Portfolio Website — Akhsan Nurramdhan

This is my personal portfolio website built to showcase my skills, projects, and experience as a web developer.  
The website serves as a central place to present my work, technical background, and contact information.

Live website: https://www.akhsannurramdhan.site

---

## About

This portfolio is built to document my growth as a developer and to provide a simple way for others to view my projects and skills.  
It is designed with a focus on clarity, responsiveness, and performance.

The goal of this project is to present myself in a structured and professional way for opportunities such as internships, freelance work, and collaboration.

---

## Features

- Responsive layout for desktop and mobile devices
- Clean and minimal interface
- Project showcase section
- About and skills section
- Contact links integration
- Deployed using Vercel with custom domain

---

## Tech Stack

- Next.js / React
- Tailwind CSS
- JavaScript / TypeScript
- HTML and CSS
- Vercel for deployment
- Git and GitHub for version control
